# MLH Code Masters

MLH Code Masters is a learning platform designed to help developers enhance their coding skills through hands-on projects, challenges, and collaborative learning.

## Features

- **Resource Library**: Access a curated library of tutorials, articles, and videos.
- **Project-Based Learning**: Work on real-world projects to build a strong portfolio.
- **Coding Challenges**: Participate in coding challenges to test and improve your skills.
- **Community Collaboration**: Join a community of developers to share knowledge and collaborate on projects.
- **Guilds**: Create or join guilds based on your interests to collaborate with like-minded developers.
- **Progress Tracking**: Monitor your learning journey and track your progress over time.
- **Certifications**: Earn certifications upon completing courses and projects.
- **MLH Points System**: Earn MLH Points for completing projects and challenges, which can be redeemed for swags and other rewards through the MLH Rewards Store.
- **Mentorship Programs**: Get guidance from experienced developers to help you grow.

## Building and Running the Application

### UI

- Install dependencies

  ```bash
    npm install
  ```

- Start the development server:

  ```bash
    npm start
  ```

- Open your browser and go to `http://localhost:3000` to access the application.

- Run lint and format checks:

  ```bash
    npm run lint:format
  ```

## Libraries and Frameworks Used

- **Frontend (UI)**:
  - React
  - Vite
  - Zustand
  - React Router
  - Axios
  - IndexedDB with Dexie.js
